# 교재 지도 (Korean textbook chapter map)

주제를 물었을 때 **어느 한국어 교재의 몇 장을 보라고 안내**하기 위한 지도.
세 권의 한국어 번역·저술서를 기준으로 한다. 원문(저작권 보호 대상)은 재수록하지 않으며, 장 구성과 주제만 정리한다.

세 교재의 성격:
- **뉴만 『네트워크』(M. Newman, *Networks*)** — 가장 포괄적·수리적. 측정→알고리즘→모형→동역학의 표준 골격.
- **바라바시 『네트워크 사이언스』(A.-L. Barabási, *Network Science*)** — 직관·시각화 중심, 척도 없는 성질·성장 모형·확산에 강점.
- **『네트워크 분석』** — 입문 친화적, 좁은 세상·허브·모델·동역학을 쉬운 순서로.

---

## 뉴만 『네트워크』 (18장)

- ch01 도입 — 네트워크 과학의 통합적 관점.
- ch02 기술 분야 네트워크 / ch03 정보 네트워크 / ch04 사회연결망 / ch05 생물학적 네트워크 — **경험적 데이터**(무엇이 노드·링크인가, 측정·편향).
- ch06 네트워크의 수학 표현 — 인접 행렬·라플라시안·방향/가중/이분.
- ch07 네트워크 측정량과 측정법 — 중심도·뭉침·전이성·모듈도·assortativity.
- ch08 컴퓨터 알고리듬 — BFS/최단경로/betweenness, 행렬·스펙트럴.
- ch09 네트워크 통계와 측정 오류 — 표집·편향·불확실성.
- ch10 실제 네트워크의 구조 — 거대 덩어리·좁은 세상·링크수 분포·커뮤니티(경험 종합).
- ch11 무작위 그래프 — G(N,p), 생성함수, 거대 덩어리.
- ch12 구조 모형 — configuration model(대표 null model).
- ch13 네트워크 형성 모형 — 선호적 연결·성장 모형.
- ch14 커뮤니티 구조 — modularity, 스펙트럴, SBM 추론.
- ch15 스미기와 네트워크 회복력 — percolation, 표적/무작위 공격.
- ch16 네트워크에서의 감염병 전파 — SIR/SIS, 감염 임계치.
- ch17 네트워크 동역학 시스템 — 동기화·동역학.
- ch18 네트워크 검색 — 분산 검색·내비게이션.

## 바라바시 『네트워크 사이언스』 (10장)

- ch01 네트워크 과학 소개 / ch02 그래프 이론 — 기초 언어.
- ch03 무작위 네트워크 — Erdős–Rényi, 거대 덩어리, 좁은 세상.
- ch04 척도 없는 성질 — power law, 허브, 링크수 지수 γ. (⭐ scale-free 질문의 1차 참조)
- ch05 바라바시–알버트 모형 — 성장 + 선호적 연결.
- ch06 변화하는 네트워크 — evolving/fitness 모형.
- ch07 링크수 상관관계 — degree correlation, assortativity.
- ch08 네트워크의 견고함 — robustness, 표적 공격, percolation.
- ch09 커뮤니티 — 커뮤니티 검출.
- ch10 확산 현상 — epidemic/spreading.

## 『네트워크 분석』 (0–7 + 부록)

- 0 서론 — 개관.
- 1 네트워크 구성요소 — 노드·링크·기본 측정량.
- 2 좁은 세상 — small world.
- 3 허브 — 척도 없는·허브.
- 4 방향성과 가중치 — directed/weighted.
- 5 네트워크 모델 — 무작위·성장 모형.
- 6 커뮤니티 — 커뮤니티 구조.
- 7 동역학 — 확산·동역학.
- 부록 — 보충.

## 사용 규칙
- 개념 설명 뒤에는 가능하면 "**더 볼 곳: [교재 장]**"을 붙여 사용자가 한국어 원서로 심화하게 한다.
- 교재 원문을 인용·재수록하지 않는다. 장·주제 안내와 자기 언어의 설명만 제공한다.
- 최신 주제(2020+)는 교재 범위 밖이므로 `최신동향.md`·`핵심문헌.md`로 안내한다.
