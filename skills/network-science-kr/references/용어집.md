# 한국어–영어 용어집 (Korean↔English glossary)

네트워크 과학 표준 용어를 한국어 번역서 기조에 맞춰 정리했다. **한국어로 설명하되 영어 원어를 병기**하고,
번역서마다 용어가 갈리는 경우 대표어 + 이형(異形)을 함께 준다. 각 항목은 관련 교재 장(`교재-지도.md` 참조)을 표시.

> 원칙: 처음 등장할 때 "한국어(English)"로 병기, 이후에는 문맥에 맞게. 수식 기호·방법명·논문명은 영어 유지.

## 기본 구성요소
- 노드(node) / 꼭짓점(vertex) — 이형: 정점. [뉴만 ch6, 사이언스 ch2]
- 링크(link) / 에지(edge) / 변 — [뉴만 ch6]
- 링크수(degree) — 한 노드에 붙은 링크 개수. 수학에선 차수. [사이언스 ch2·3]
- 링크수 분포(degree distribution) p_k — [사이언스 ch3·4]
- 평균 링크수(average degree) ⟨k⟩ — [사이언스 ch3]
- 인접 행렬(adjacency matrix) A — [뉴만 ch6]
- 방향성 네트워크(directed network), 가중치 네트워크(weighted network) — [뉴만 ch6, 분석 ch4]
- 이분 네트워크(bipartite network) — [뉴만 ch6]

## 거리·경로·뭉침
- 경로(path), 최단 경로(shortest path), 거리(distance), 지름(diameter) — [뉴만 ch6, 사이언스 ch2]
- 뭉침 계수(clustering coefficient) C — 이형: 군집 계수, 뭉치기 계수. [사이언스 ch2, 뉴만 ch7]
- 전이성(transitivity) — 삼각형 밀도 기반 전역 뭉침. [뉴만 ch7]
- 좁은 세상(small world) — 짧은 경로 + 높은 뭉침. [분석 ch2, 사이언스 ch3]

## 척도 없는 성질·허브
- 허브(hub) — 링크수가 매우 큰 노드. [분석 ch3, 사이언스 ch4]
- 척도 없는 네트워크(scale-free network) — 링크수 분포가 거듭제곱. [사이언스 ch4]
- 거듭제곱 법칙(power law) / 멱법칙 — p_k ~ k^(-γ). [사이언스 ch4]
- 링크수 지수(degree exponent) γ — [사이언스 ch4]

## 모형(생성 모형)
- 무작위 네트워크(random network), 에르되시–레니(Erdős–Rényi) 모형 G(N,p) — [사이언스 ch3, 뉴만 ch11]
- 구조 모형(configuration model) — 링크수 분포를 보존한 무작위화(대표 null model). [뉴만 ch12]
- 선호적 연결(preferential attachment) — 이형: 우선적 연결. [사이언스 ch5]
- 바라바시–알버트(Barabási–Albert) 모형 — 성장 + 선호적 연결. [사이언스 ch5]
- 변화하는/진화하는 네트워크(evolving network) — [사이언스 ch6, 뉴만 ch13]

## 상관·섞임
- 링크수 상관관계(degree correlation) — [사이언스 ch7]
- 끼리끼리 섞임(assortative mixing) / 동류성(assortativity) — [뉴만 ch7, 사이언스 ch7]

## 중심도(centrality)
- 중심도(centrality) / 중심성 — 노드 중요도. "중요도"는 하나가 아님. [뉴만 ch7]
- 링크수 중심도(degree centrality) — 국소적. [뉴만 ch7]
- 사이 중심도(betweenness centrality) — 이형: 매개 중심성. 흐름/중개. [뉴만 ch7]
- 근접 중심도(closeness centrality) — 도달성. [뉴만 ch7]
- 고유벡터 중심도(eigenvector centrality), 카츠 중심도(Katz), 페이지랭크(PageRank) — 재귀적 영향. [뉴만 ch7]

## 커뮤니티(community)
- 커뮤니티(community) / 모듈(module) — [사이언스 ch9, 뉴만 ch14, 분석 ch6]
- 모듈도(modularity) Q — 이형: 모듈성. configuration-model 기대치와의 차이. [뉴만 ch14]
- 커뮤니티 검출(community detection) — 이형: 커뮤니티 탐지. [뉴만 ch14]
- 확률적 블록 모형(stochastic block model, SBM) — 추론적 커뮤니티. [뉴만 ch14 관련]
- 루뱅(Louvain)·라이덴(Leiden)·인포맵(Infomap) — 알고리즘 이름은 원어 유지.

## 견고함·스미기·동역학
- 견고함(robustness) — 이형: 강건성. [사이언스 ch8, 뉴만 ch15]
- 회복력(resilience) — [뉴만 ch15]
- 스미기(percolation) — 이형: 침투. 거대 덩어리 출현 전이. [뉴만 ch15]
- 거대 덩어리(giant component) / 거대 연결 요소 — [사이언스 ch3, 뉴만 ch11]
- 덩어리(component) / 연결 요소 — [뉴만 ch6]
- 감염병 전파(epidemic spreading) / 확산 현상(spreading phenomena) — [사이언스 ch10, 뉴만 ch16]
- SIR·SIS 모형, 감염 임계치(epidemic threshold) — [뉴만 ch16]
- 동기화(synchronization), 동역학 시스템(dynamical systems) — [뉴만 ch17]
- 네트워크 검색(network search) — [뉴만 ch18]

## 수학·스펙트럼
- 라플라시안(Laplacian) L, 정규화 라플라시안 — [뉴만 ch6·8]

## 사용 규칙
- 용어가 갈리면 **대표어를 쓰되 괄호로 이형과 영어를 병기**해 혼선을 막는다.
- 사용자가 특정 번역서를 따르면 그 책 용어에 맞춘다(예: 뉴만 번역서는 "모듈도", "스미기").
- 수학 기호·알고리즘명·논문명은 번역하지 않는다.
